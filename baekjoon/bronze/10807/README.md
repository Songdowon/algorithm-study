# 🧩 BOJ 10807 - 개수 세기

## 📌 문제 정보
- **난이도**: Bronze 5
- **링크**: https://www.acmicpc.net/problem/10807

---

## 📝 문제 요약
- N개의 정수가 주어진다  
- 특정 값 v가 몇 번 등장하는지 출력하는 문제

---

## 💡 핵심 아이디어
- 리스트에서 특정 값의 개수를 세면 된다  
- `count()` 또는 반복문으로 해결 가능

---

## 🧠 접근 과정
- 처음 생각:
  - 리스트를 순회하면서 직접 개수 세기
- 막힌 부분:
  - 없음 (기초 구현 문제)
- 해결 과정:
  - `list.count(v)`를 사용해 간단하게 해결

---

## 🛠 사용한 것
- 리스트 (`list`)
- `count()` 함수 
- 입력 처리 (`input().split()`)

---

## ⚠️ 주의할 점
- 입력이 문자열로 들어오므로 `int` 변환 필요  
- 공백 기준으로 나누기 위해 `split()` 사용

---

## 📚 배운 점
- Python에서는 `count()` 같은 내장 함수로 코드 간결하게 작성 가능  
- 기본 자료구조 활용이 중요
