# 🧩 BOJ 2292 - 벌집

## 📌 문제 정보
- **난이도**: Bronze 2
- **링크**: https://www.acmicpc.net/problem/2292

---

## 📝 문제 요약
- 방이 벌집 모양으로 퍼져 있다
- 시작 방은 `1`번이고, 중심에 위치한다
- 그 다음 층부터는 방이 다음과 같이 늘어난다
  - 2번째 층: 6개
  - 3번째 층: 12개
  - 4번째 층: 18개
- 즉, 한 층이 늘어날 때마다 방 개수가 `6`씩 증가한다
- 어떤 방 번호 `N`이 주어졌을 때,
  **1번 방에서 N번 방까지 이동할 때 거치는 방의 수**를 구하는 문제이다

---

## 💡 핵심 아이디어
- 벌집은 바깥층으로 갈수록 방 개수가 `6, 12, 18, ...`처럼 증가한다
- 각 층의 마지막 방 번호를 누적해서 구하면 된다
- `1 → 7 → 19 → 37 → ...`처럼 범위를 넓혀 가면서
  입력값 `N`이 어느 층에 속하는지 찾으면 된다
- 즉, **층의 범위를 직접 누적하는 방식**으로 해결할 수 있다

---

## 🧠 접근 과정
- 처음에는 좌표를 직접 구해야 하는 문제인지,
  규칙을 식으로 정리해야 하는 문제인지 고민했다
- 하지만 실제 좌표를 구할 필요는 없고,
  각 층의 마지막 번호만 알면 된다는 점을 발견했다
- 규칙을 확인해 보면:
  - 1층: `1`
  - 2층: `1 + 6 = 7`
  - 3층: `7 + 12 = 19`
  - 4층: `19 + 18 = 37`
- 즉, 현재 층의 마지막 번호에 `6 × (현재 층 번호)`를 계속 더해 가면 된다
- 입력한 `N`이 현재 범위 안에 들어오는 순간,
  그 층 번호가 정답이다

---

## 🛠 사용한 것
- `while`문
- 정수 덧셈
- 누적합
- 조건문 `if`

---

## ⚠️ 주의할 점
- 이 문제는 **방 번호의 위치를 직접 찾는 문제처럼 보여도**,
  실제로는 **층의 범위만 구하면 되는 문제**이다
- `N = 1`인 경우는 예외처럼 보이지만,
  사실 처음 층이므로 정답은 바로 `1`이다
- 층이 늘어날 때마다 증가하는 방 개수는
  `6, 12, 18, 24, ...`처럼 **6의 배수**라는 점을 놓치지 않아야 한다
- 방 개수를 하나씩 세는 방식으로 접근하면 비효율적이고,
  **층 단위로 범위를 확장**하는 방식이 더 적절하다

---

## ✅ 배운 점
- 문제를 풀 때는 무작정 계산하기보다,
  먼저 **문제 안의 규칙**을 파악하는 것이 중요하다
- 규칙이 있는 수열 문제는 직접 구조를 구현하기보다
  **증가 패턴을 먼저 정리하는 것**이 더 중요하다
- 정확한 점화식을 세우는 것보다,
  이 문제를 해결할 수 있는 알고리즘을 구현하는 방향으로 생각하게 되었다
- 반복문으로 범위를 누적해 가면서
  어떤 값이 어느 구간에 속하는지 찾는 문제를 연습할 수 있었다
- 복잡한 좌표 계산 없이도
  **패턴만 찾으면 단순 반복으로 해결 가능**하다는 점을 배울 수 있다

## 🔄 여러 구현 방식 비교

| 방법 | 특징 |
|------|------|
| 범위 직접 누적 ✅ | 가장 직관적이고 많이 쓰임 |
| 마지막 방 번호 기준 갱신 | 현재 층의 끝 번호를 기준으로 판단 |
| `N == 1` 예외 처리 후 반복 | 시작값을 분리해서 생각하기 쉬움 |
