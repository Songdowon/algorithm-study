# 🧩 BOJ 2903 - 중앙 이동 알고리즘

## 📌 문제 정보
- **난이도**: Bronze 3
- **링크**: https://www.acmicpc.net/problem/2903

---

## 📝 문제 요약
- 정사각형 모양의 점 무늬가 있다
- 처음에는 한 변에 점이 `2개` 있어서 전체 점 개수는 `4개`이다
- 이동을 한 번 할 때마다 각 변의 점 사이에 새로운 점들이 추가된다
- `N`번 이동한 뒤 전체 점의 개수를 구하는 문제이다

---

## 💡 핵심 아이디어
- 한 변의 점 개수가 어떻게 변하는지 보면 일정한 규칙이 있다
- 처음 한 변의 점 개수는 `2`
- 한 번 이동할 때마다  
  `이전 한 변의 점 개수 × 2 - 1`
  이 된다
- 따라서 `N`번 후 한 변의 점 개수는 $2^N + 1$
- 전체 점 개수는 정사각형이므로 $(2^N + 1)^2$ 를 계산하면 된다

---

## 🧠 접근 과정
- 처음에는 점을 직접 그려 보면서 규칙을 찾는다
- 전체적인 규칙이 보이지 않으면, 한 변의 점 개수처럼 더 단순한 부분의 패턴부터 살펴본다
- 이렇게 찾은 패턴을 바탕으로 전체 점 개수의 규칙을 정리한다

---

## 🛠 사용한 것
- 거듭제곱 연산 `**`
- 정수 계산
- 반복문
- 점화식 아이디어

---

## ⚠️ 주의할 점
- 점의 **전체 개수**를 바로 구하려고 하기보다  
  먼저 **한 변의 점 개수**를 구하는 것이 훨씬 쉽다
- 전체 점 개수는 정사각형 구조이므로  
  반드시

$$
(\text{한 변의 점 개수})^2
$$

로 계산해야 한다
- 규칙을 직접 몇 번 써 보면 공식을 더 쉽게 찾을 수 있다

---

## ✅ 배운 점
- 규칙 찾기 문제에서는  
  **작은 입력을 직접 써 보면서 패턴을 찾는 과정**이 중요하다
- 전체를 한 번에 보지 말고  
  **한 변의 점 개수**처럼 더 단순한 대상으로 바꿔 생각하면 문제를 쉽게 풀 수 있다
- 반복문으로도 풀 수 있고
  규칙을 찾으면 **수식 하나로 더 간단하게** 해결할 수 있다

---

## 🔄 여러 구현 방식 비교

| 방법 | 특징 |
|------|------|
| 공식 사용 ✅ | 가장 간단하고 빠름 |
| 반복문으로 한 변 점 개수 갱신 | 규칙 이해에 도움이 됨 |
| 전체 점 개수를 점화식처럼 갱신 | 가능하지만 직관성은 조금 떨어짐 |
