# 🧩 BOJ 2566 - 최댓값

## 📌 문제 정보
- **난이도**: Bronze 3
- **링크**: https://www.acmicpc.net/problem/2566

---

## 📝 문제 요약
- 9×9 크기의 숫자 표가 주어진다
- 이 표 안에서 **가장 큰 값**을 찾는다
- 그 최댓값이 **몇 번째 행, 몇 번째 열**에 있는지도 함께 출력한다
- 행과 열 번호는 **1부터 시작**한다

---

## 💡 핵심 아이디어
- 2차원 리스트를 입력받는다
- 전체 원소를 하나씩 확인하면서 현재까지의 최댓값과 위치를 저장한다
- 더 큰 값을 발견하면 최댓값과 행, 열 위치를 갱신한다
- 최종적으로 최댓값과 위치를 출력한다

---

## 🧠 접근 과정
- 처음 생각:
  - 모든 값을 하나씩 비교하면서 가장 큰 값을 찾으면 된다
- 핵심:
  - 숫자만 찾는 것이 아니라 **위치(행, 열)**도 같이 저장해야 한다
- 해결:
  - 이중 `for`문으로 전체를 순회하면서
    최댓값과 그 좌표를 함께 갱신한다

---

## 🛠 사용한 것
- 2차원 리스트 `[][]`
- 이중 `for`문
- 조건문 `if`
- 최댓값 비교
- 행/열 인덱스 저장
- `max()`

---

## ✅ 내가 배운 점
- 2차원 리스트에서는 값을 탐색할 때 **행과 열을 함께 관리**해야 한다
- 2차원 배열 입력은  
  `matrix = [list(map(int, input().split())) for _ in range(9)]`  
  처럼 작성하면 한 줄씩 입력받아 리스트 안에 리스트 형태로 저장할 수 있다
-`matrix[i][j]`는 `i`행 `j`열의 값을 의미한다
- 문제에서 행, 열 번호가 1부터 시작하면  
  파이썬 인덱스 `0` 기준에서 **출력할 때 +1 보정**이 필요하다
- `return`은 여러 개의 값을 동시에 반환할 수 있고, 여러 변수로 나누어 받을 수도 있다

---

## 🔄 여러 구현 방식 비교

| 방법 | 특징 |
|------|------|
| 이중 `for`문 ✅ | 가장 기본적이고 위치 추적이 쉬움 |
| 리스트 펼치기 | 값과 위치를 튜플로 저장해 처리 가능 |
| `max()` 활용 | 최댓값만 구하기 간단하지만 위치 찾기는 추가 작업 필요 |
