# 🧩 BOJ 10818 - 최소, 최대

## 📌 문제 정보
- **난이도**: Bronze 3
- **링크**: https://www.acmicpc.net/problem/10818

---

## 📝 문제 요약
- N개의 정수가 주어진다  
- 이 중에서 **최솟값과 최댓값을 출력**해야 한다  

---

## 💡 핵심 아이디어
- 전체 값을 순회하면서 **최솟값과 최댓값을 갱신**
- 또는 Python의 **`min()`, `max()` 함수 사용**

---

## 🧠 접근 과정
- 처음 생각:
  - 모든 값을 리스트에 저장 후 `min()`, `max()` 사용
- 확장:
  - 반복문으로 직접 최솟값/최댓값 구하기
- 추가 최적화:
  - `map`을 사용해 **메모리 절약 방식으로 처리 가능**

---

## 🛠 사용한 것
- `min(), max()`
- 반복문 (`for`)
- `map(int, input().split())`
- `float('inf')` (초기값 설정)

---

## ⚠️ 주의할 점
- 초기값을 0으로 설정하면 **음수 입력에서 오류 발생 가능**
- `map` 객체는 **한 번만 순회 가능**

---

## 📚 배운 점
- 같은 문제도 **여러 방식으로 해결 가능**
- Python 내장 함수(`min`, `max`)는 매우 강력함
- 메모리 효율을 고려한 **iterator 방식 처리(map)** 이해
