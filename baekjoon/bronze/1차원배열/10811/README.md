# 🧩 BOJ 10811 - 바구니 뒤집기

## 📌 문제 정보
- **난이도**: Bronze 2  
- **링크**: https://www.acmicpc.net/problem/10811  

---

## 📝 문제 요약
- 1번부터 N번까지 번호가 적힌 바구니가 일렬로 놓여 있음  
- M번의 작업이 주어지고, 각 작업은 구간 `[i, j]`를 뒤집는 것  
- 모든 작업 수행 후 바구니 상태를 출력  

---

## 💡 핵심 아이디어
- 리스트의 **부분 구간 뒤집기**가 핵심
- Python에서는 슬라이싱으로 쉽게 해결 가능  
- `basket[i-1:j] = basket[i-1:j][::-1]`

---

## 🧠 접근 과정
- 처음 생각:
  - 직접 swap하면서 뒤집기 (투포인터 방식)
- 더 좋은 방법:
  - Python 슬라이싱 + `[::-1]` 사용하면 한 줄로 해결 가능
- 추가 확장:
  - `reversed()`, 직접 리스트 생성 등 다양한 방식으로 구현 가능

---

## 🛠 사용한 것
- `list`
- `slicing`
- `[::-1]`
- `reversed()`
- `tuple` (입력 저장)
- 반복문 (`for`, `while`)

---

## 📚 배운 점

- slicing을 이용한 구간 처리
  - 리스트에서 특정 구간을 자르는 방법
  - `i-1` 하는 이유 → **1-based → 0-based 인덱스 변환**

- [::-1] 
  - `[::-1]`은 리스트를 뒤집는 가장 간단한 방법
  - 내부적으로는 **step = -1** (`start:end:step`)
    - step이 음수 → 뒤에서부터 읽기
  - 가장 효율적인 방법

- tuple 사용 → `(i, j)` 형태의 **변경되지 않는 데이터 묶음**
  - 리스트보다 **의미적으로 더 명확함**
