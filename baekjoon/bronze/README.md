## 📂 Category Details

### 1️⃣ 1차원 배열
- 배열 기본 사용
- 인덱스 처리
- 반복문 기반 구현
- 슬라이싱 이용한 구간 처리

---

### 2️⃣ 문자열
- 문자열 기본 사용 (immutable 특징)
- 인덱싱 / 슬라이싱
- 문자열 ↔ 숫자 변환 ('int', 'str')
- 문자열 메서드 활용 ('find', 'count', 'replace' 등)
- 문자 처리 ('ord', 'chr')
- 문자열 순회 ('for', 'enumerate')
- 문자열 매핑 (문자 → 값 변환)
- 자료구조 선택 (list, set, dict)과 시간복잡도를 고려한 구현

---

### 3️⃣ 2차원 배열
- 이중 반복문 (`for`)
- 2차원 리스트 입력 받기
- 문자열 / 리스트 → 2차원 해석
- data[i][j] - i번째 행, j번쨰 열
- 행 기준 / 열 기준 탐색
- 좌표(row, col) 개념
- list comprehension
- `zip()`을 활용한 같은 위치 묶기

---

### 4️⃣ 수학
- 문제 해결을 위한 **규칙 찾기**
- 작은 예시를 직접 써 보며 **패턴 확인**
- 규칙을 **점화식 / 수식**으로 정리
- 반복문으로 **범위 확장 / 누적합 계산**
- **그리디 방식** 등 규칙 기반 접근 
- `//`, `%`, `**`등 **파이썬 연산 활용**
- **부동소수점 오차를 피하기 위해 정수 계산 우선**
