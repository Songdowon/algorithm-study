# 🧩 BOJ 2562 - 최댓값 + 위치 찾기

## 📌 문제 정보
- **난이도**: Bronze 3
- **링크**: https://www.acmicpc.net/problem/2562

---

## 📝 문제 요약
- 9개의 정수가 주어진다  
- 이 중에서 **최댓값과 그 위치(몇 번째 수인지)**를 출력한다  

---

## 💡 핵심 아이디어
- 값을 순회하면서 **최댓값과 해당 위치를 동시에 갱신**
- `enumerate()`를 사용하면 **값 + 위치를 함께 처리 가능**

---

## 🧠 접근 과정
- 처음 생각:
  - 리스트에 저장 후 `max()`로 최댓값, index로 해당 위치 구하기

- 개선:
  - `enumerate()`를 활용해 **위치까지 한 번에 처리**

---

## 🛠 사용한 것
- `enumerate()`
- `max()`
- 반복문 (`for`)
- list comprehension

---

## ⚠️ 주의할 점
- 인덱스 값이 아닌 **위치가 1부터 시작**해야 함
  → `enumerate(..., start=1)` 사용해야 함

---

## 📚 배운 점
- `enumerate()` 를 활용하면 인덱스와 값을 동시에 다룰 수 있다.
- `max(~, key= )` 으로 기준을 정해서 최댓값을 출력할 수 있다.
  → `key=`: **정렬/비교** 관련 함수에만 있는 매개변수
