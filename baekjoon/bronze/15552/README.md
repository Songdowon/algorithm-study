# 🧩 BOJ 15552 - 빠른 A+B

## 📌 문제 정보
- **난이도**: Bronze 4
- **링크**: https://www.acmicpc.net/problem/15552

---

## 📝 문제 요약
- 입력 개수가 매우 많다  
- 여러 줄에 걸쳐 두 정수를 입력받아 합을 출력해야 한다  
- 일반적인 `input()` 사용 시 시간 초과 발생 가능

---

## 💡 핵심 아이디어
- `input()` 대신 `sys.stdin.readline()` 사용  
- 빠른 입력 처리를 통해 시간 초과 방지  

---

## 🧠 접근 과정
- 처음 생각:
  - 일반적인 A+B 문제처럼 `input()` 사용
- 막힌 부분:
  - 입력 개수가 많아 시간 초과 발생
- 해결 과정:
  - `sys.stdin.readline()`으로 변경하여 해결

---

## 🛠 사용한 것
- `sys.stdin.readline`
- 반복문 (`for`)
- `map(int, input().split())`

---

## ⚠️ 주의할 점
- `input()` 사용 시 시간 초과 발생 가능  
- `sys.stdin.readline()`은 문자열 끝에 개행(`\n`) 포함  
  → 필요 시 `strip()` 사용 고려

---

## 📚 배운 점
- Python에서는 입력 데이터가 많을 때  
  **입출력 방식 자체가 성능에 큰 영향을 준다**
- 알고리즘이 단순해도 **입출력 최적화가 중요**
