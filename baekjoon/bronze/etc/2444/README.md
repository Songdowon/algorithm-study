# 🧩 BOJ 2444 - 별 찍기 - 7

## 📌 문제 정보
- **난이도**: Bronze 3
- **링크**: https://www.acmicpc.net/problem/2444

---

## 📝 문제 요약
- 입력으로 정수 `N`이 주어진다  
- 총 `2N - 1`개의 줄에 걸쳐 별(`*`)을 출력한다  
- 위쪽은 별이 점점 늘어나고, 아래쪽은 점점 줄어드는 **다이아몬드 형태**를 만든다

---

## 💡 핵심 아이디어
- 각 줄마다 **공백 개수**와 **별 개수**의 규칙을 찾는다  
- 위쪽과 아래쪽을 나누어 출력하면 구현이 쉽다  
- `i`번째 줄의 별 개수는 `2 * i - 1` 형태로 증가한다

---

## 🧠 접근 과정
- 처음 생각:
  - 별을 하나씩 찍어야 하나 고민함
  - 반복문으로 수식을 어떻게 세팅할지 고민함
- 핵심 발견:
  - 각 줄마다 공백과 별의 개수가 일정한 규칙을 가짐
- 해결 과정:
  - 위쪽 삼각형과 아래쪽 역삼각형으로 나누어 출력
  - 공백은 `' ' * 개수`, 별은 `'*' * 개수`로 문자열 생성

---

## 🛠 사용한 것
- 반복문 (`for`)
- 문자열 곱셈 (`' ' * k`, `'*' * k`)
- 출력 패턴 설계

---

## ⚠️ 주의할 점
- 아래쪽 출력은 가운데 줄이 중복되지 않도록  
  `n - 1`부터 시작해야 한다
- 공백과 별의 개수를 정확히 계산해야  
  대칭이 맞는 모양이 출력된다

---

## 📚 배운 점
- 별찍기 문제는 단순 출력이 아니라 **반복문과 규칙성을 찾는 연습**이다
- 파이썬의 메서드나 모듈 활용도 중요하지만,
  **함수와 로직을 깔끔하게 설계하는 것**의 중요성을 체감했다 
- 문자열 곱셈을 사용하면 복잡한 출력도 간단하게 구현할 수 있다
- 대칭 구조를 위쪽 / 아래쪽으로 나누면 훨씬 직관적으로 풀 수 있다
