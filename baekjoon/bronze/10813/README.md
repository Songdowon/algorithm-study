# 🧩 BOJ 10813 - 공 바꾸기

## 📌 문제 정보
- **난이도**: Bronze 2
- **링크**: https://www.acmicpc.net/problem/10813

---

## 📝 문제 요약
- N개의 바구니가 있다 (초기 상태: 1번부터 N번까지 공이 들어 있음)
- M번의 입력이 주어진다
- 각 입력마다 i번 바구니와 j번 바구니의 공을 서로 교환한다
- 최종 바구니 상태를 출력한다

---

## 💡 핵심 아이디어
- 초기 상태를 `[1, 2, ..., n]`형태의 리스트로 생성
  두 위치의 값을 바꾸는 **swap(교환) 연산** 반복
- 인덱스는 1부터 시작하므로 → 0-based index로 변환 필요(`i-1`, `j-1`) 

---

## 🧠 접근 과정
- 처음 생각: 
  - 바구니를 리스트로 만들고 두 값을 반환
- 구현:
  - swap을 이용하여 값 교환
- 확장:
  - 임시 변수(temp)을 이용한 방식으로도 구현 가능

---

## 🛠 사용한 것
- 리스트 초기화 (`list(range(1, n + 1))`)
- 반복문 (`for`)
- `map(int, input().split())`
- swap(`a, b = b, a`) 또는 temporary variable

---

## ⚠️ 주의할 점
- 인덱스가 1부터 시작하므로 **반드시 `i-1`, `j-1`로 변환해야 함**
- swap 시 값이 덮어쓰이지 않도록 주의(`temp` 또는 tuple 방식 사용)

---

## 📚 배운 점
- Python에서는 `a, b = b, a`로 간단하게 swap 가능
- swap은 내부적으로 **tuple을 이용한 동시 할당**으로 동작함
- 같은 문제도 여러 방식(직접 구현, 내장 기능)으로 구현 가능
